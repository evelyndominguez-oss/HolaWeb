MAP

Permite recorrer un arreglo y crear uno nuevo transformando cada elemento con una función

FILTER

Sirve para filtrar los elementos de un arreglo que cumplen una condición, y devuelve un nuevo arreglo solo con ellos.

REDUCE

Toma todos los valores de un arreglo y los reduce a un solo resultado, como sumar o multiplicar.

FIND

Busca y devuelve el primer elemento que cumple una condición.

SORT

Sirve para ordenar los elementos de un arreglo, ya sea de forma numérica o alfabética.

INCLUDES

VVerifica si un arreglo contiene un valor específico, y devuelve true o false

INDEX OF

Indica la posición (índice) donde se encuentra un elemento dentro del arreglo..

SOME

Devuelve true si al menos uno de los elementos cumple la condición..

EVERY

Devuelve true si todos los elementos cumplen la condición.

SLICE

Copia una parte específica del arreglo sin modificar el original.
